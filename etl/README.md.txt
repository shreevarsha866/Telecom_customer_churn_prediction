# Customer Churn ETL Pipeline (SQL Server)

## Overview
End-to-end ETL pipeline for customer churn analysis using SQL Server.

## Architecture
CSV → Staging Table → Transformation → Final Table → Analytics View

## Tools
- SQL Server (SSMS)
- CSV Flat File Import
- GitHub

## ETL Flow
1. Create staging & final tables
2. Load CSV into staging
3. Transform & cleanse data
4. Create analytical views
5. Perform data quality checks

## Output
- Curated churn dataset
- Churn rate by contract type
